What is AIUML?

AIUML (AI-Optimized UML) is an open-source experiment in improving AI-assisted development. Instead of having LLMs explore your code file-by-file, AIUML provides a pre-computed architectural "map" that may help them understand your system's structure and relationships.

Status: Experimental Software - This is a preview release exploring new approaches. Results vary based on codebase complexity and LLM capabilities.

The Concept

The Observation

When working with unfamiliar code, LLMs typically:

  • Read files sequentially to understand the system
  • May revisit the same relationships repeatedly
  • Sometimes miss important architectural patterns
  • Use significant context exploring rather than solving

Our Approach

AIUML attempts to provide LLMs with:

  • Pre-extracted class hierarchies and relationships
  • Method signatures and visibility information
  • Module organization and dependencies
  • A structured format designed for text-based consumption

Potential Benefits

Based on early testing, AIUML may help with:

Reducing time spent discovering code relationships
Maintaining architectural consistency in suggestions
Identifying potential impacts across components
Improving context efficiency (token reduction is a side effect)

Note: Results vary significantly based on codebase complexity and LLM capabilities

Example Output

Standard Format

UserService < BaseService {
  +authenticate(username: str, password: str): User
  -validate_credentials(): bool
  => UserRepository [uses]
  => CacheService [depends on]
}

Compressed Format (High)

UserServiceUserRepository[uses]=>CacheService[depends on]}

Compression reduces tokens while preserving all architectural information. Common types are abbreviated (s=string, b=boolean, i=integer).

  • Architecture: Design and relationships (recommended)
  • API: Public interfaces with documentation
  • Full: Complete extraction for deep analysis

Compression Options

Scale Handling

Powerful SQL Query Capability

AIUML stores your project structure in a SQLite database, enabling powerful custom queries:

# Find all classes with more than 500 lines
aiuml query "SELECT name, file_path, (line_end - line_start) as lines
             FROM classes WHERE (line_end - line_start) > 500"

# Find methods that might need refactoring
aiuml query "SELECT c.name, m.name, m.complexity
             FROM methods m JOIN classes c ON m.class_id = c.id
             WHERE m.complexity > 10"

# Analyze language distribution
aiuml query "SELECT language, COUNT(*) as count
             FROM classes GROUP BY language"

Query your codebase like a database - find patterns, identify issues, and gain insights that would be difficult to discover otherwise.

An Experimental Approach

AIUML is based on a hypothesis: that providing LLMs with structured architectural knowledge might improve their effectiveness when working with code. This is an experiment, not a proven solution.

What We're Exploring

  • Whether pre-computed architecture helps more than exploration
  • If structured formats improve LLM code comprehension
  • How compression affects understanding vs token efficiency
  • Which extraction settings work best for different scenarios

What We Don't Claim

  • Specific performance improvements
  • Guaranteed error reduction
  • Universal effectiveness
  • That this approach works for all codebases or LLMs

Frequently Asked Questions

Is AIUML right for my project?

AIUML might be worth trying if you work with LLMs for development, have medium to large codebases, want to experiment with improving AI assistance, and are comfortable with experimental tools.

How much does it cost?

Nothing. AIUML is free open-source software (MIT license) and always will be.

Does AIUML send my code anywhere?

No. AIUML itself runs entirely on your machine and never transmits data. However, when you share the extracted model with an LLM, you're choosing to send that architectural information to your LLM provider.

What languages are supported?

Python is fully tested. JavaScript, TypeScript, Java, C++, C#, and PHP have experimental support that's continuously improving.

Will this definitely improve my LLM's performance?

No guarantees. AIUML is an experiment. Results vary based on your codebase, the LLM you're using, and your specific use case.

Can I contribute?

Yes! We welcome contributions, feedback, and testing. Check our GitHub repository for contribution guidelines.
